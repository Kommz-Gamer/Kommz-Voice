# Kommz Voice - Message Guidelines

Use this framing consistently across GitHub, Patreon, Discord, and website pages.

## Short positioning

- Kommz Gamer Community = open-source client layer
- Kommz Voice = backend engine layer (XTTS + GPT-SoVITS)
- Discord = user and contributor community
- Patreon = support and member tiers

## Suggested one-liner

`Kommz Gamer Community is the open client, while Kommz Voice powers the speech engine layer (XTTS + GPT-SoVITS).`

## Suggested paragraph

`The ecosystem is split clearly: the Community repository is open-source and focused on client usage, while Kommz Voice handles the speech engine backend (XTTS and GPT-SoVITS). Discord centralizes community feedback and support, and Patreon helps sustain development and member-only channels.`

## Avoid ambiguous wording

Do not imply that:
- only the client exists
- Patreon replaces GitHub
- Discord is only for premium users

Preferred wording:
- open-source on GitHub
- engine backend in Kommz Voice
- Discord for everyone
- Patreon for support and tier-based extras
